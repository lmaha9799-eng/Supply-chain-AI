# Intelligent Supply Chain & Demand Forecasting Dashboard

An end-to-end analytics platform that structures corporate retail transaction records in a SQL database and
applies a trend-momentum mathematical algorithm to predict upcoming inventory demands.

## 🛠️ How to Run the Project Local
1. Install dependencies:
   `pip install streamlit pandas matplotlib`
2. Initialize and populate the SQL Database:
   `python setup_db.py`
3. Launch the visual web dashboard:
   `streamlit run dashboard.py`
